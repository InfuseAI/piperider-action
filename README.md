# PipeRider CLI

An action for PipeRider CLI to collaboratively develop pipelines and manage data incidents.

# Usage

See [action.yml](action.yml)


```yaml
steps:
...
- uses: InfuseAI/piperider-action@v0.1.0
  input:
    stages: stages/*.yaml
```
